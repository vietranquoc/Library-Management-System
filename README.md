# Library-Management-System
Library Management System - Full-stack pet project (Angular 17 + Spring Boot 3)  \n\nFeatures:  \n• Unique book ID, multi-authors, multiple copies  \n• Search by title/author/genre/year  \n• Borrow, reserve &amp; auto 7-day fine  \n• Librarian dashboard: borrowing history, member status &amp; fines  Learning-focused project mastering Angular + Spring Boot.
